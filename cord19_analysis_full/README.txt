CORD-19 Research Analysis Project
---------------------------------
This project includes:
- analysis.py : Performs data cleaning and creates visualizations
- app.py      : A Streamlit web app to explore the dataset interactively
- data/metadata.csv : A sample dataset (~500 rows)

Instructions:
1. Install dependencies:
   pip install pandas matplotlib wordcloud streamlit

2. To run the analysis script:
   python analysis.py

3. To launch the Streamlit dashboard:
   streamlit run app.py
